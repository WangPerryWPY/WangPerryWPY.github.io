发表于 2019-07-04 分类于 数据结构 评论数： 阅读次数：
本文字数： 7.9k
  • 主要实现的难点在于稀疏矩阵的转置以及矩阵乘法。
  • 对用到的数据结构定义不懂的可参考严蔚敏老师的数据结构书籍，这里讲严老师讲的两种数据结构三元组结构 和行逻辑结构 整合到一起。
  • 稀疏矩阵是线性结构中比较难的一块知识点，不懂的建议去b站看看严老师的视频。
  • 实现环境：linux
